GET: http://notebook/users gets all users 
GET: http://notebook/users/id  get user by id
POST: http://notebook/users creating new user 
POST: http://notebook/users/id  update user by id
DELETE: http://notebook/users/id  delete user by id


GET: http://notebook/records  gets all records 
GET: http://notebook/records/id  get record by id
POST: http://notebook/records creating new record 
POST: http://notebook/records/id  update record by id
DELETE: http://notebook/records/id  delete record by id
